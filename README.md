# KUDWA™ Focus

[![build_windows](https://github.com/NLorenzoD/kudwa_focus/actions/workflows/build_windows.yml/badge.svg)](https://github.com/NLorenzoD/kudwa_focus/actions/workflows/build_windows.yml)

KUDWA Focus is a full-screen Windows countdown timer built for consultants, facilitators and workshop rooms. It turns breaks and timed activities into visible, energetic shared moments that help participants return together.

## Download

1. Open the [latest GitHub release](https://github.com/NLorenzoD/kudwa_focus/releases/latest).
2. Download `kudwa_focus.exe`.
3. Double-click it on a 64-bit Windows 10 or Windows 11 computer.

The app is self-contained, so a separate .NET installation is not required. Because this personal release is not commercially code-signed, Microsoft Defender SmartScreen may show a warning. Choose **More info**, verify that the file came from this repository, and then choose **Run anyway**.

## Built-in timers

- Break — 10 minutes
- Tea Break — 15 minutes
- Lunch Break — 60 minutes
- Exercise — 5 minutes
- Quick Exercise — 3 minutes
- Custom Activity — any name, from one second to 240 minutes

## The experience

- A projector-scale countdown with sharp KUDWA-inspired angular motion.
- Gentle original music throughout the break.
- An automatically intensifying final-minute score.
- A ten-impact countdown and large cinematic finish.
- Start, pause, reset, add a minute and subtract a minute.
- Full-screen mode, mute and adjustable volume.
- No account, database, analytics or network access.

## Keyboard controls

| Key | Action |
| --- | --- |
| `Space` | Start or pause |
| `F11` | Enter or exit full screen |
| `Esc` | Exit full screen |
| `M` | Mute or unmute |
| `R` | Reset |
| `+` / `-` | Add or subtract one minute |

## Original audio

The two WAV files embedded in the executable are an original procedural score generated by [`tools/generate_audio.py`](tools/generate_audio.py). Their durations, hashes and provenance are recorded in [`audio_manifest.json`](src/kudwa_focus/assets/audio/audio_manifest.json).

Regenerate them with:

```bash
python3 tools/generate_audio.py
```

## Build locally

Requirements: Windows 10 or 11 and the .NET 10 SDK.

```powershell
dotnet run --project src/kudwa_focus/kudwa_focus.csproj
```

Create the self-contained executable:

```powershell
dotnet publish src/kudwa_focus/kudwa_focus.csproj --configuration Release --runtime win-x64 --self-contained true --output dist
```
